API 文档

BoltSearch 提供 RESTful API，所有端点基于 http://localhost:8080/api。

端点列表

方法路径说明
POST/api/index上传 JSONL 文件，自动索引并去重
POST/api/docs添加单篇文档，重复返回 409
GET/api/search全文搜索，支持 BM25 评分
GET/api/docs/:id获取指定文档
DELETE/api/docs/:id删除文档及关联索引
GET/api/stats数据库统计信息
GET/api/suggest词项自动补全
GET/api/browse浏览 Bucket 列表或内容

POST /api/index

上传 .jsonl 文件批量索引。请求类型 multipart/form-data，字段名 file。自动跳过重复文档。

curl -F "file=@data.jsonl" http://localhost:8080/api/index

响应: {"indexed": 10, "summary": "（新增 10 篇）"}

POST /api/docs

添加单篇文档。请求体 application/json。

curl -X POST http://localhost:8080/api/docs \
  -H "Content-Type: application/json" \
  -d '{"title":"标题","content":"正文"}'

成功 201: {"docID": 11} | 重复 409: {"error":"文档已存在","duplicate":true}

GET /api/search

全文搜索。支持 BM25 评分排序。

参数默认说明
q必填查询关键词
modeor搜索模式: or / and
limit10返回条数
offset0分页偏移
prefixfalse启用前缀匹配
 
curl "http://localhost:8080/api/search?q=处理器&mode=and&prefix=true&limit=5"

GET /api/suggest

词项自动补全，基于索引 Bucket 前缀扫描。

参数默认说明
prefix必填前缀
limit20返回数量

GET /api/browse

无参数时返回所有 Bucket 列表（name/desc/count）。指定 ?bucket=name 时返回 Bucket 内容表格（headers + rows 数组）。

curl "http://localhost:8080/api/browse?bucket=docs&limit=50"

GET /api/stats

返回 TotalDocs / TotalTokens / AvgDocLen / UniqueTerms / DBFileSize。

DELETE /api/docs/:id

删除指定文档，自动清理倒排索引、文档频率、去重哈希。

curl -X DELETE http://localhost:8080/api/docs/5

前端交互说明

本页面使用 HTMX（部分交互）和原生 Fetch API 与 BoltSearch 后端通信。页面加载时自动调用 POST /api/index 导入 10 篇示例数据。搜索框实时调用 GET /api/suggest 获取补全建议，提交时调用 GET /api/search。右侧栏通过 GET /api/browse 加载 Bucket 列表，点击 Bucket 调用 GET /api/browse?bucket= 查看详细表格数据。添加和删除分别调用 POST /api/docs 和 DELETE /api/docs/:id。头部统计每 5 秒轮询 GET /api/stats 更新。
